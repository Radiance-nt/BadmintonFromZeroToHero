## 基础学习路线

1. 正手高远球，*原地高远球，蹬转跳挥，后退击球高远球（步伐）*
2. 正手挑球
3. 正手发球
4. 反手挑球
5. 反手发球
6. 后场两边高远球（步伐）
7. 正反手网前挑球（步伐）
8. 正反手网前放网（步伐）
9. 全场四点（步伐）
10. 正手吊球
11. 正手杀球
12. 两边接杀
13. 两边抽球
14. 全场六点（步伐）
15. 反手过渡（步伐）
16. 反手高原球（步伐）
