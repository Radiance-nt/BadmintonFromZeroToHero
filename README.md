可以证明，在忽略空气阻力的情况下，出球角度为45度时射程最大。而有空气阻力时，初速度越小，羽毛球受空气阻力影响越小，最大射程角越接近于45度 。初速度越大，最大射程角越小。

13. 两边抽球
